### 문제 설명

<p>이 문제는 <strong>투 스텝</strong> 문제이다. 한 번 채점할 때 참가자의 프로그램을 2번 실행하는 형식의 문제이다. 각 실행은 일반적인 문제와 같이 진행되지만, 두 실행은 서로 독립적이다.</p>

<p>쿠옹이는 단웅이에게 금고의 비밀번호를 전달하려 한다. 금고의 비밀번호는 $6$개의 $1$ 이상 $64$ 이하인 정수로 구성되어 있다.</p>

<p>하지만 비밀번호를 직접 전해주면 악당이 비밀번호를 중간에 뺏을 수 있으니 쿠옹이는 비밀번호를 암호화해 전달하려 한다. 암호화된 비밀번호 역시 $1$ 이상 $64$ 이하의 정수 $6$개로 구성되어야 하며, 원래 비밀번호에 포함되는 수의 집합을 $S_{A}$, 암호화된 비밀번호에 포함되는 수의 집합을 $S_{B}$라고 할 때 $S_{A} \cap S_{B} = \emptyset$를 만족해야 한다.</p>

<p>단웅이는 이렇게 쿠옹이가 암호화한 비밀번호를 복호화해 원래 비밀번호를 알아내려 한다. 둘은 암호화 및 복호화 전략을 미리 상의해 정해놓을 수 있다.</p>

### 입력 

 <p>첫 줄에 $t$가 주어진다. $t = 1$이면 쿠옹이의 암호화 전략을 수행해야 하며, $t = 2$이면 단웅이의 복호화 전략을 수행해야 한다. ($1 \le t \le 2$)</p>

<p>$t = 1$이면, 다음 줄에 원래 비밀번호인 $6$개의 정수 $A_{1}$, $A_{2}$, $A_{3}$, $A_{4}$, $A_{5}$, $A_{6}$이 공백으로 구분되어 주어진다. ($1 \le A_{i} \le 64$)</p>

<p>$t = 2$이면, 다음 줄에 암호화된 비밀번호인 $6$개의 정수 $B_{1}$, $B_{2}$, $B_{3}$, $B_{4}$, $B_{5}$, $B_{6}$이 공백으로 구분되어 주어진다. ($1 \le B_{i} \le 64$)</p>

### 출력 

 <p>$t = 1$이면, $S_{A} \cap S_{B} = \emptyset$을 만족하는 암호화된 비밀번호인 $6$개의 정수 $B_{1}$, $B_{2}$, $B_{3}$, $B_{4}$, $B_{5}$, $B_{6}$를 공백으로 구분하여 출력한다. ($1 \le B_{i} \le 64$)</p>

<p>$t = 2$이면, 복호화된 비밀번호인 $6$개의 정수 $C_{1}$, $C_{2}$, $C_{3}$, $C_{4}$, $C_{5}$, $C_{6}$을 공백으로 구분하여 출력한다. ($1 \le C_{i} \le 64$)</p>

<p>$1 \le i \le 6$을 만족하는 모든 정수 $i$에 대하여 $A_{i} = C_{i}$면 <span style="color:#16a085;"><strong>맞았습니다!!</strong></span>를 받는다.</p>
