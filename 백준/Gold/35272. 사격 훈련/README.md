### 문제 설명

<p>하늘이와 푸르매는 사격 훈련을 하게 되었다.</p>

<p>사격 훈련은 두 번의 기록 사격으로 이루어지며, 두 번의 사격 중 더 높은 점수가 최종 점수가 된다. 각 기록 사격은 $N$발의 총알을 자신의 사로에 쏘며, 명중한 총알의 개수가 해당 기록 사격의 점수가 된다.</p>

<p>첫 번째 기록 사격 결과 하늘이는 만발을 쐈고, 푸르매는 한 발도 맞히지 못했다. 이미 최고 점수를 기록한 하늘이는 두 번째 기록사격에서 푸르매의 사로에 총을 몇 발 쏘아 푸르매의 점수를 높여주기로 했다. 그러나 표적지에 $N$발보다 많은 총알이 명중했다면 부정 행위로 처리되어 $0$점을 받게 된다!</p>

<p>또한 사격 도중에는 표적지에 명중한 총알의 개수를 알 수 없으므로, 하늘이는 본인과 푸르매의 명중률을 가지고 푸르매의 사로에 쏠 적절한 총알 수를 결정해야 한다. 즉, 푸르매가 받는 최종 점수의 기댓값을 최대화해야 한다.</p>

<p>하늘이의 명중률은 $P$, 푸르매의 명중률은 $Q$이며 푸르매는 자신에게 주어진 총알 $N$발을 모두 자신의 표적지에 겨냥해 쏜다고 할 때, 푸르매가 얻게 될 점수의 기댓값의 최댓값을 구하여라.</p>

### 입력 

 <p>첫번째 줄에 $N$이 주어진다. ($1 \leq N \leq 20$)</p>

<p>두번째 줄에 $P$, $Q$가 공백으로 구분되어 주어진다. ($0 < P, Q < 1$)</p>

### 출력 

 <p>푸르매가 얻게 될 점수의 기댓값의 최댓값을 출력한다.</p>

<p>정답과의 절대 오차 또는 상대 오차가 $10^{-9}$ 이하이면 정답으로 처리된다.</p>
