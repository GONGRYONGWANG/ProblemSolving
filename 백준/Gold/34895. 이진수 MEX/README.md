### 문제 설명

<p>길이가 $N$인 이진수열 $S$가 주어진다. $S$의 연속 부분수열<sup>[1]</sup>이 <strong>아닌</strong> 이진수들 중 가장 작은 것을 찾아보자.</p>

<p>$0$으로 시작하는 이진수는 없다고 가정한다.</p>

<hr>
<p><sup>[1]</sup> 길이 $N$인 수열 $A = [A_1, A_2, \cdots, A_N]$에서 어떤 $1\leq L \leq R \leq N$에 대해 $[A_L, A_{L+1}, \cdots, A_R]$ 형태로 나타나는 부분수열을 $A$의 연속 부분수열이라 한다.</p>

### 입력 

 <p>첫째 줄에 정수 $N$이 주어진다. $(1 \leq N \leq 200\,000)$</p>

<p>둘째 줄에 길이가 $N$이고 $0$과 $1$로만 이뤄진 이진수열 $S$가 주어진다. $(|S|=N)$</p>

### 출력 

 <p>첫째 줄에 답을 출력한다.</p>
