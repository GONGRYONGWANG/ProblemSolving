<p>메이플 용사로서 루비 금고 이벤트에 참가한 팬텀 역시 더 많은 상품을 원하고 있었다. 비밀번호가 모두 입력된 것을 본 팬텀은, 메이플 월드 최고의 괴도답게 비밀번호를 조작해서 받게 될 상품 수를 최대한 크게 만드는 작전을 세웠다.</p>
