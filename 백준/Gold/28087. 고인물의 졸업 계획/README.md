### 문제 설명

<p>SCSC에는 아직 졸업하지 못한 고인물이 있다. 학교에 너무 오래 다니다보니 모든 것이 시시해진 고인물은 새로운 자극을 찾아 졸업 계획을 세우기로 했다. 졸업 계획을 세우기 위해 <a href="https://sugang.snu.ac.kr/">졸업 시뮬레이션</a>을 실행해 본 고인물은 졸업을 하려면 최소 <mjx-container class="MathJax" jax="CHTML" style="font-size: 109%; position: relative;"><mjx-math class="MJX-TEX" aria-hidden="true"><mjx-mi class="mjx-i"><mjx-c class="mjx-c1D441 TEX-I"></mjx-c></mjx-mi></mjx-math><mjx-assistive-mml unselectable="on" display="inline"><math xmlns="http://www.w3.org/1998/Math/MathML"><mi>N</mi></math></mjx-assistive-mml><span aria-hidden="true" class="no-mathjax mjx-copytext">$N$</span></mjx-container>학점을 더 수강해야 한다는 사실을 알아내었다. 또한, 행정상 한 학기에 수강 가능한 최대 학점은 <mjx-container class="MathJax" jax="CHTML" style="font-size: 109%; position: relative;"><mjx-math class="MJX-TEX" aria-hidden="true"><mjx-mn class="mjx-n"><mjx-c class="mjx-c32"></mjx-c></mjx-mn><mjx-mi class="mjx-i"><mjx-c class="mjx-c1D441 TEX-I"></mjx-c></mjx-mi></mjx-math><mjx-assistive-mml unselectable="on" display="inline"><math xmlns="http://www.w3.org/1998/Math/MathML"><mn>2</mn><mi>N</mi></math></mjx-assistive-mml><span aria-hidden="true" class="no-mathjax mjx-copytext">$2N$</span></mjx-container>이다.</p>
