### 문제 설명

<p>호현이와 수아는 지하철 게임을 하고 있다.</p>

<p>지하철 게임이란, 특정 호선이 호명되면 해당 호선에 속한 지하철 역 이름을 중복되지 않게 부르고, 더 이상 역 이름을 말할 수 없게 되는 경우 패배하는 게임이다.</p>

<p>지하철 노선을 모두 외워 버려 게임에 흥미를 잃은 호현이는, 다음과 같이 규칙을 개정할 것을 제안하였다.</p>

<ul>
<li>역명을 모두 역 번호로 대체한다. 한 호선에 $N$개의 역이 있는 경우, 역 번호는 $1, 2, \dots, N$까지 있다.</li>
<li>앞 사람이 말한 역 번호에서 $-2, -1, 1, 2$ 중 하나를 골라 더한 역 번호만 말할 수 있다. 단, 게임을 시작할 때에 선공 플레이어는 임의의 역 번호를 말할 수 있다.</li>
<li>게임은 항상 단일 호선 내에서만 일어나며, 다른 호선으로의 환승은 이루어지지 않는다.</li>
<li>더 이상 역 번호를 말할 수 없는 경우 패배한다.</li>
</ul>

<p>호현이가 선공 플레이어, 수아가 후공 플레이어이고, 각 플레이어는 항상 최선의 선택을 할 때, 게임을 이기는 사람은 누구인지 맞혀 보자!</p>

### 입력 

 <p>첫 번째 줄에 게임의 판 수 $T$가 주어진다. ($1 \leq T \leq 100$)</p>

<p>두 번째 줄부터 $T$개의 줄에 걸쳐 각 호선에 속한 역의 개수 $N$이 주어진다. ($1 \leq N \leq 10^9$)</p>

### 출력 

 <p>$T$개의 줄에 걸쳐, 각 게임에 대해 호현이가 이긴 경우 <code><span style="color:#e74c3c;">H</span></code>를, 수아가 이긴 경우 <code><span style="color:#e74c3c;">S</span></code>를 한 줄에 하나씩 출력한다.</p>
