### 문제 설명

<p>짐비가 <a href="/problem/34977">수미수열</a> 문제를 세팅하던 중 어디선가 날아온 거북이가 데이터를 망가뜨렸다. 대회가 얼마 남지 않아 패닉이 온 짐비는 손상된 데이터를 최대한 살려보려고 한다.</p>

<p><a href="/problem/34977">수미수열</a> 문제의 손상된 데이터와 의도한 출력이 주어질 때, 손상된 값에 임의의 숫자를 넣어 의도한 출력을 만들도록 데이터를 재구성하자.</p>

### 입력 

 <p>첫번째 줄에 수열의 길이 $N$이 주어진다. $(2 \leq N \leq 1\,000)$</p>

<p>두번째 줄에 손상된 수열 $L_1, \ldots , L_N$이 공백으로 구분되어 주어진다. $L_i=0$일 경우 $i$번째 항이 손상되었음을, 그렇지 않은 경우 손상되지 않았음을 의미한다. $(0 \leq L_i \leq 2\,000)$</p>

<p>세번째 줄에 손상되기 전의 데이터가 정해를 통해 나왔던 출력 $S$가 주어진다. $S$는 <code><span style="color:#e74c3c;">yes</span></code> 혹은 <code><span style="color:#e74c3c;">no</span></code>이다.</p>

<p>손상된 항은 하나 이상 존재한다. 또한 주어지는 입력은 모순되지 않는다. 즉, 손상된 값을 $2\,000$ 이하의 양의 정수로 바꿔 출력 결과가 $S$가 되게 하는 경우는 항상 존재한다.</p>

### 출력 

 <p>첫번째 줄에 손상된 부분을 채워넣은 수열을 <a href="/problem/34977">수미수열</a> 문제의 정해에 넣었을 때 출력 결과가 $S$가 되는 $1$ 이상 $2\,000$ 이하의 정수로만 이루어진 수열을 출력한다.</p>

<p>가능한 출력이 여러가지라면 그중 아무거나 하나를 출력한다.</p>
