### 문제 설명

<blockquote>
<p>직행은 취업 과정에서 발생하는 불필요한 복잡함을 줄이기 위해 만들어진 채용 서비스이다. 여러 채용 사이트에 흩어져 있는 채용 공고를 대규모 크롤링으로 수집하고, 자연어 처리(NLP) 기술을 활용해 플랫폼별, 직무별로 정리하여 제공하고 있다. 이를 통해 구직자는 반복적인 검색과 비교 과정 없이, 자신에게 맞는 채용 공고를 보다 빠르게 탐색할 수 있다.</p>

<p>이러한 방식으로 설립 1년 만에 연간 방문자 수 100만 명을 달성하며, 국내 채용 서비스 기준 상위 5위권에 진입했다. 직행은 소프트웨어와 데이터 기술을 기반으로 취업 준비 과정을 단순화하고, 궁극적으로 ‘직장인의 행복’에 기여하는 것을 목표로 한다.</p>
</blockquote>
