### 문제 설명

<blockquote>
<p>"난 벨만-포드보다 데이크스트라가 더 좋아...</p>

<p>근데 데이크스트라보다 BFS가 더 좋은데 어떡해?"</p>
</blockquote>

<p>일반적으로, 간선에 가중치가 있는 무향 그래프에서 최단 경로는 간선 가중치의 합이 최소인 경로를 의미한다. 하지만 BFS를 너무 좋아하는 재원이는 경로에 포함된 간선의 개수를 최소화하고 싶다.</p>

<p>간선의 개수만 최소화하는 모든 경로를 <strong data-end="268" data-start="259">레몬 경로</strong>라고 정의한다. <strong>레몬 경로의 가중치</strong>는 해당 경로에 포함된 간선들의 가중치 합을 의미하며, <strong>레몬 거리</strong>는 두 정점을 잇는 모든 <strong>레몬 경로</strong>에 대한 <strong>레몬 경로의 가중치</strong>의 평균을 의미한다.</p>

<p>정점이 $N$개이고, 양방향 간선이 $M$개 있는 연결 그래프가 주어진다. $i$번째 간선은 $U_i$와 $V_i$를 연결하며, 그 가중치는 $C_i$이다.</p>

<p>$1$번 정점과 다른 모든 정점 사이의 <strong data-end="454" data-start="441">레몬 거리</strong>를 구하여라. 그래프에는 자기 자신으로 가는 간선이나 같은 두 정점을 잇는 여러 간선이 존재할 수 있다.</p>

### 입력 

 <p>입력은 다음과 같은 형식으로 주어진다.</p>

<blockquote>
<p>$N \ M$</p>

<p>$U_1 \ V_1 \ C_1$</p>

<p>$U_2 \ V_2 \ C_2$</p>

<p>$\vdots$</p>

<p>$U_M \ V_M \ C_M$</p>
</blockquote>

### 출력 

 <p>첫째 줄부터 $N-1$개의 줄에 걸쳐 $i$번째 줄에 $1$번 정점과 $i+1$번 정점 사이의 레몬 거리를 다음과 같은 방법으로 출력한다.</p>

<p>레몬 경로의 길이가 $\dfrac{p}{q}$일 경우 $qm\equiv p \ (\mathrm{mod} \ 998 \ 244 \ 353)$, $0\le m<998 \ 244 \ 353$인 양의 정수 $m$을 출력한다. 단, $p$와 $q$는 서로소인 양의 정수이다. 또한 레몬 경로의 개수가 $998 \ 244 \ 353$의 배수인 경우가 주어지지 않음이 보장된다.</p>
