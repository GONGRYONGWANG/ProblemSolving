### 문제 설명

<p>이루매는 이번 기말고사 실험을 시작했다. U, O, S 세가지 재료로 UOS 세트를 만드는 실험이였다. 이루매가 처음에 가진 재료는 U가 $U$개, O가 $O$개, S가 $S$개이다. 이 때, UOS 하나를 만들기 위해서는 U, O, S가 1개씩 필요하다. 그런데 갑자기 이루매가 억지를 부리기 시작했다.</p>

<blockquote>
<p>U는 너무 많잖아! 이거 두 개랑 S 하나랑 바꿔줘!</p>
</blockquote>

<p>이루매의 억지부리기에 어쩔 수 없이 이루매가 U를 $2$개 줄 때마다 S $1$개로 바꿔주기로 했다. 이제 이루매는 U를 마음껏 S로 바꿀 수 있게 되었고, 한 번 바꿀 때마다 이루매가 가진 U는 $2$개 줄고, S는 $1$개 늘어난다. 이때, 이루매가 만들 수 있는 UOS 세트의 최대 개수는 얼마일까?</p>

### 입력 

 <p>첫째 줄에 이루매가 처음에 가진 U, O, S의 재료의 수를 의미하는 정수가  공백으로 구분되어 주어진다. ($ 0 \le U, O, S \le 10^{18} $)</p>

### 출력 

 <p>첫째 줄에 이루매가 만들 수 있는 UOS 세트의 최대 개수를 출력한다.</p>
