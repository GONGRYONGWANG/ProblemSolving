### 문제 설명

<p>기숙사에서 잠을 자다가, 오늘 자료구조 강의가 있다는 사실을 방금 깨달은 달구는 급하게 기숙사를 뛰쳐나왔다.</p>

<p>원래 기숙사에서 강의실까지 걷는 경우 $t$초가 걸리지만, 열심히 달려가면 이동 시간을 $k$% 만큼 줄일 수 있다. 예를 들어, 달리기로 이동 시간이 $10$% 줄어든다고 하자. 걸어서 강의실까지 가는 데 $100$초가 걸린다면, 달리는 경우 달구는 $90$초 만에 강의실에 도착할 수 있다.</p>

<p>현재 시각과 오늘 있는 강의의 시작 시각이 주어질 때, 달구가 강의 시작 전까지 강의실에 도착할 수 있는지 판단하시오.</p>

### 입력 

 <p>첫째 줄에 현재 시각이 주어진다.</p>

<p>둘째 줄에 강의 시작 시각이 주어진다.</p>

<p>모든 시각은 <code>HH:MM:SS</code> 형식으로 주어진다. ($0 \le$ <code>HH</code> $\le 23$; $0 \le$ <code>MM</code> $\le 59$; $0 \le$ <code>SS</code> $\le 59$)</p>

<p>셋째 줄에 강의실까지 걸어갈 때 걸리는 시간 $t$와 달릴 때 이동 시간이 줄어드는 비율 $k$가 공백으로 구분되어 정수로 주어진다. ($0 \le k \le 100$; $100 \le t \le 10\,000$; $t$는 $100$의 배수)</p>

### 출력 

 <p>달구가 달려서 강의 시작 시각까지 강의실에 도착할 수 있다면 <span style="color:red;"><code>1</code></span>을, 아니라면 <span style="color:red;"><code>0</code></span>을 출력한다. 강의 시작 시각에 정확히 강의실에 들어와도 도착할 수 있는 것으로 판단한다.</p>
