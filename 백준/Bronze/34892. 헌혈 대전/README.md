### 문제 설명

<p>얼마 전 시립대, 경희대, 외대 사이에서 헌혈 대전이 열렸다. 세 대학교의 헌혈 참가자 수의 총합은 $N$명이다. 헌혈 대전에서 집계를 맡은 용준은 대전의 승자를 바로 알려주지 않고, 다음과 같은 힌트만 남기고 떠났다.</p>

<blockquote>
<p>시립대, 경희대, 외대의 참가자 수를 각각 $X, Y, Z$라고 할 때, 다음 두 식이 성립한다.</p>

<p>$\begin{cases} A X + B Y + C Z = D \\ E X + F Y + G Z = H \end{cases} $</p>
</blockquote>

<p>용준이 남긴 힌트를 바탕으로 세 학교의 참가자 수 $X, Y, Z$를 구해보자.</p>

### 입력 

 <p>정수 $N, A, B, C, D, E, F, G, H$가 공백으로 구분되어 주어진다. $(0 \le N \le 10\,000; 0 \le A, B, C, D, E, F, G, H \le 100\,000)$</p>

### 출력 

 <p>답이 하나만 존재한다면, 첫째 줄에 0을 출력한다. 그리고 둘째 줄에 음이 아닌 정수 $X, Y, Z$를 공백으로 구분하여 출력한다.</p>

<p>답이 여러 개 존재한다면, 첫째 줄에 1을 출력한다.</p>

<p>답이 존재하지 않는다면, 첫째 줄에 2를 출력한다.</p>
