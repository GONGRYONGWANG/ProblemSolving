### 문제 설명

<p>시을이가 가장 좋아하는 문자열은 <span style="color:#e74c3c;"><code>SciComLove</code></span>입니다. 시을이는 <span style="color:#e74c3c;"><code>SciComLove</code></span>가 너무 좋았던 나머지, 양의 정수 $x$, $y$에 대해 <span style="color:#e74c3c;"><code>SciCom</code></span>을 $x$번 반복한 뒤, <span style="color:#e74c3c;"><code>Love</code></span>를 $y$번 반복한 문자열 $S$를 동아리실에 전시해 두었습니다. 예를 들어, $x=1$이고 $y=2$일 경우, $S$는 <span style="color:#e74c3c;"><code>SciComLoveLove</code></span>입니다.</p>

<p>어느 날, 악당 도윤이가 문자열 $S$의 문자 순서를 뒤죽박죽 섞었습니다. 불쌍한 시을이를 위해 뒤죽박죽 섞인 문자열을 보고 $S$를 복원해 봅시다. 단, 대소문자가 다르면 서로 다른 문자로 취급합니다.</p>

### 입력 

 <p>첫째 줄에 도윤이가 뒤죽박죽 섞은 문자열이 주어집니다.</p>

### 출력 

 <p>첫째 줄에 도윤이가 문자 순서를 섞기 전의 원래 문자열 $S$를 출력합니다.</p>
