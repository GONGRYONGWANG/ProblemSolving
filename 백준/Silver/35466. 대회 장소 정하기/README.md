### 문제 설명

<p>둘레의 길이가 $N$인 원형 도로 위에 세 학교 UNIST, DGIST, POSTECH이 있다.</p>

<p>원형 도로 위에 있는 두 지점 간의 <strong>거리</strong>는, 시계 방향으로만 이동하는 경로와 반시계 방향으로만 이동하는 경로의 길이 중 더 작은 것으로 정의한다. 이때, UNIST와 DGIST 사이의 거리는 $a$, DGIST와 POSTECH 사이의 거리는 $b$, 그리고 POSTECH과 UNIST 사이의 거리는 $c$이다.</p>

<p>윤이, 달구, 포닉스는 UDPC의 합동 온사이트 대회 개최를 위해 기나긴 회의 끝에 대회를 열 장소를 정했다. 대회장의 위치는 다음과 같은 특징을 가진다.</p>

<ul>
<li>대회장의 위치는 원형 도로 위의 한 지점이다. 대회장의 위치가 각 학교의 위치와 동일할 수도 있다.</li>
<li>UNIST와 대회장과의 거리는 $p$이다.</li>
<li>DGIST와 대회장과의 거리는 $q$이다.</li>
</ul>

<p>그렇다면 POSTECH과 대회장과의 거리는 얼마일까?</p>

### 입력 

 <p>첫째 줄에 원형 도로의 둘레 길이 $N$이 주어진다. $(3\le N\le 100)$</p>

<p>둘째 줄에 각 학교 사이의 거리를 의미하는 $a,b,c$가 공백으로 구분되어 주어진다. $(1\le a,b,c< {N \over 2})$</p>

<p>셋째 줄에 대회장과 UNIST, DGIST 사이의 거리 $p,q$가 공백으로 구분되어 주어진다. $(0\le p,q\le {N \over 2})$</p>

<p>입력에서 주어지는 수는 모두 정수이고, 주어진 거리 조건을 만족하는 각 학교의 위치와 대회장의 위치가 존재하는 입력만 주어진다.</p>

### 출력 

 <p>첫째 줄에 POSTECH과 대회장과의 거리를 출력한다. 주어진 입력 조건 하에서 답이 유일하게 결정됨을 증명할 수 있다.</p>
