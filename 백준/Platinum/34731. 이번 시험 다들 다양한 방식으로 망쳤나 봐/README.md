### 문제 설명

<p>마침내 중간고사가 끝났다.</p>

<p>준영이의 친구들 $N$명이 서로 점수를 비교해보고 있다.</p>

<p>$M$개의 비교 결과가 다음과 같이 주어진다.</p>

<ul>
<li>$x$ $y$: $y$번 친구의 점수는 $x$번 친구의 점수보다 크거나 같다. 즉, $score[y] \ge score[x]$ 이다.</li>
</ul>

<p>$X$번 친구인 강민이는 다음과 같이 자랑하려고 한다.</p>

<blockquote>
<p>이번 시험 다들 다양한 방식으로 망쳤나 봐. 내 점수보다 낮은 <strong>서로 다른 점수값</strong>이 $t$개나 있는 것 같아!</p>
</blockquote>

<p>강민이는 자신의 똑똑함을 강조하기 위해, 관측된 $M$개의 비교 결과들과 모순되지 않는 선에서 $t$를 가장 크게 말하려 한다.</p>

<p>가능한 $t$의 최댓값을 구하시오.</p>

### 입력 

 <p>첫째 줄에 $N$ $M$ $X$가 주어진다. $(1\leq N,M \leq 200\,000 ; 1\leq X \leq N)$</p>

<p>둘째 줄부터 $M$개 줄에 걸쳐 관계를 나타내는 $x, y$가 공백으로 구분되어 주어진다. $(1\leq x,y \leq N)$</p>

<p>주어지는 모든 수는 정수이다.</p>

### 출력 

 <p>가능한 $t$의 최댓값을 출력한다.</p>
