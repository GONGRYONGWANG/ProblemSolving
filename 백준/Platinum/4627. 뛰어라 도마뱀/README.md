### 문제 설명

<p>용재는 애완 도마뱀들과 함께 던전을 돌다가 이상한 방에 입장했다. 건질 게 없나 주위를 둘러보던 용재는 뒤를 돌아보고 깜짝 놀랐다. 막내 도마뱀이 함정을 밟아 방의 바닥이 사라져버린 것이다! 밑에서는 불길이 올라오기 시작했고, 도마뱀들은 바닥이 사라지고 남은 기둥 위에 서 있다.</p>
