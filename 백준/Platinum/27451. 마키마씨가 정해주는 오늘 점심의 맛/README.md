<p>마음씨가 착한 마키마 씨는 이들을 위해 셋이 만날 수 있는 식당을 찾아주려고 한다. 마키마 씨를 도와 셋이 만날 수 있는 식당을 찾아주자. 이동하는 동안 점심시간은 계속 지나기 때문에, 셋이 최대한 빨리 식사를 할 수 있게 이동하는 경로의 길이가 가장 짧은 식당을 찾아주자.</p>
