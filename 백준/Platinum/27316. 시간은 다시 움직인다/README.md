### 문제 설명

<p>한별이는 어느 날 골동품 화살에 찔린 이후 초능력자가 되어 시간을 조종하는 신비한 힘, “백금 한별”을 얻게 되었다. 능력을 사용하는 동안에는 한별이가 시간을 다르게 느끼기 때문에, 마치 시간이 멈춘 것 같은 느낌을 받게 된다.</p>

<p>PS 애호가이기도 했던 한별이는, 이를 PS 문제를 푸는 것에 이용하고자 한다.</p>
