### 문제 설명

<p>어느 날 SNUPC 부원들은 최고의 알고리즘을 정하는 토론을 하게 되었다. 논쟁 끝에 최고의 알고리즘은 기하 혹은 정수론 중 하나라는 사실에 모두가 동의할 수 있었지만, 기하를 지지하는 부원과 정수론을 지지하는 부원 간의 의견 차는 도저히 좁혀지지 않았다. 이대로라면 싸움이 끝이 없을 것이기에 이들은 SNUPC 게임을 통해 승부를 가르기로 했다.</p>

<p>SNUPC 게임은 다음과 같이 진행된다.</p>

<ol>
	<li>첫 번째 칸에 S가, 두 번째 칸에 N, 세 번째 칸에 U, 네 번째 칸에 P, 다섯 번째 칸에 C가 적혀져 있는 $5$칸짜리 보드를 준비한다.</li>
	<li>$0$부터 $10^8$까지 수가 하나씩 적혀 있는 주사위를 $5$번 던져, 각 주사위의 결과에 해당하는 개수만큼 $5$개의 칸에 각각 말을 놓는다.</li>
	<li>두 부원이 번갈아가며, 각자의 턴에 말을 이동시킨다. 자신의 턴에 말을 이동시키지 못한다면 패배한다.</li>
</ol>

<p>다음 중 하나의 방법으로 말을 이동시킬 수 있다.</p>

<ol>
	<li>C가 아닌 문자가 적혀 있는 칸의 말 중 하나를 잡아 보드의 다음 칸으로 움직인다. S가 적힌 칸에 말이 있는 경우 N이 적힌 칸으로 움직이며, N에서 U로, U에서 P로, P에서 C로 움직인다.</li>
	<li>U가 적힌 칸에 있는 말 하나를 잡아 S로 움직인다.</li>
</ol>

<p>두 부원들은 자존심이 강한 천재들이기 때문에, 항상 최선의 방법으로 말을 이동시키고, 지는 것보다는 게임이 영원히 지속되는 것을 선호한다.</p>

<p>기하를 지지하는 부원이 먼저 행동을 시작할 때, 승리하는 부원을 출력하라. 만약 영원히 승부가 끝나지 않는다면, 게임 이론이 최고의 알고리즘이라고 생각하는 한별이가 승리하게 된다!</p>

### 입력 

 <p>첫 번째 줄에 테스트 케이스의 개수 $T$가 주어진다. ($1\le T\le 10^5$)</p>

<p>두 번째 줄부터 한 줄에 하나씩 각 테스트 케이스의 입력이 차례대로 주어진다.</p>

<p>각 테스트 케이스는 한 줄로 구성되어 있으며, 주사위를 던져 나온 수를 의미하는 $S$, $N$, $U$, $P$, $C$의 정수 $5$개가 공백으로 구분되어 주어진다. ($0\le S,N,U,P,C\le 10^8$) 즉, 두 부원들이 말을 움직이기 시작할 때 S가 적힌 칸에는 $S$개, N이 적힌 칸에는 $N$개, U가 적힌 칸에는 $U$개, P가 적힌 칸에는 $P$개, C가 적힌 칸에는 $C$개의 말이 존재한다.</p>

### 출력 

 <p>각 테스트 케이스 마다 기하를 지지하는 부원이 승리하면 <code>geometry</code>, 정수론을 지지하는 부원이 승리하면 <code>number_theory</code>, 영원히 승부가 끝나지 않는다면 <code>hanbyeol</code>을 출력한다.</p>
