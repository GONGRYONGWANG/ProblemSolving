### 문제 설명

<p>$N$개의 구간 $[s_i, e_i]$과 값 $v_i$가 존재한다. 이 때, $f(l, r, n)$을 $l \le s_i \le e_i \le r$인 모든 $i$에 대한 $n^{v_i}$의 곱이라 하자.</p>

<p>$1 \le l \le r \le M$인 모든 가능한 $[l, r]$ 구간에 대해, $f(l, r, r-l+1)$의 곱을 $10^9+7$로 나눈 나머지를 구하여라.</p>

### 입력 

 <p>첫 번째 줄에 $N$과 $M$이 차례대로 주어진다. ($1 \le N, M \le 10^6$)</p>

<p>두 번째 줄부터 $N$개의 줄에 걸쳐, 각 줄에 $s_i$, $e_i$, $v_i$가 차례대로 주어진다. ($1 \le s_i \le e_i \le M$, $1 \le v_i \le 10^6$)</p>

<p>입력으로 주어지는 수는 모두 정수이다.</p>

### 출력 

 <p>첫 번째 줄에 답을 출력한다.</p>
